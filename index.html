CAPITULO 8 TESTANDO TEXTOS

TAGS HTML

Obs: todas as tags serão utilizadas com parênteses angulares - < e >

Negrito = strong
ex: texto em NEGRITO

Italico = em
ex: texto em ITALICO

Marcar textos = mark
ex: texto MARCADO

Entre linhas = small
ex: TEXTO EM entrelinhas

Texto deletado ou tachado = del
ex: esse texto está TACHADO ou DELETADO

Texto sublinhado = ins
ex: texto SUBLINHADO

Texto sobrescrito = sup
ex: preciso de 20m3 de àgua na caixa

Texto subescrito = sub
ex: àgua em química é H2O

Considerar espaços, estruturas de textos e etc = pre
ex: sem exemplos

A tag que formata seu texto para ficar com espaçamentos iguais e fonte única
ex: n1 = int(input('Insira um numero inteiro'))

CAPITULO 9 - LISTAS

LISTAS ORDENADAS

Para as listas ordenadas teremos como tag OL cujo o seu
significado é - Listas Ordenadas e como seu próprio
nome diz, ele cria uma lista com ordenamento.
ex:

  1. Acordar cedo
  2. ir trabalhar
  3. voltar para casa
  4. dormir

        Toda a formatação segue se pela tag OL e em seguida identado
        segue-se a tag LI que cujo significado é diretamente lista e
        onde será preenchido a lista
        ex:
        
          <ol type="1" start="1">
            <li> Seu item de lista n°1 </li>
            <li> Seu item de lista n°2 </li>
            <li> Seu item de lista n°3 </li>
          <\ol>
        
        Contudo podendo alterar o tipo de enumeração de lista e a ordem
        pela qual poderá começar utilizando os parâmetros type e o start
      
        Seguindo o mesmo exemplo acima porém com algumas alterações,
        no caso do UL, cujo seu significado quer dizer lista não ordenada,
        nesse caso seguimos o mesmo exemplo acima somente com uma alteração
        em que no caso do type ter somente numeros, letras e algarismo romano
        nesse caso tem as opções como circulo preenchido, circulo sem
        preenchimento e quadrado para poder organizar as listas.
